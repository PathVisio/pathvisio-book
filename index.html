ğŸ“ About PathVisio

This current version of PathVisio 4.0.0 builds upon the 3.x architecture and libGPML.

Graphical Pathway Markup Language (GPML) is the native file format used store and edit pathway models in PathVisio and share them on WikiPathways.

Please check out New Features for information and links for updates.
